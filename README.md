# ❤️ 왕예민 애인과 연애하기

<프로불편러와 연애하기>는 아주 예민한 성격의 애인에게 이별 통보를 당하지 않도록 대화를 이어가야 하는 **텍스트 기반 연애 시뮬레이션 게임** 이다.<br><br>
애인은 매우 예민하기 때문에 다음과 같은 상황이 발생하면 즉시 이별을 선언한다.
<br>
- 답장이 너무 늦을 때
- 성의 없는 단답을 보낼 때
- 너무 많이 웃거나 물음표를 많이 사용할 때
- 부정적인 말투를 사용할 때 (⭐️ 감성 분석 모델 사용)
<br>

---

## 🤖 사용 모델

```alsgyu/sentiment-analysis-fine-tuned-model``` <br>
<br>
AI 허브 한국어 감정 데이터셋을 활용하여 Hugging Face의 beomi/KcBERT-base 모델을 파인튜닝한 감정 분석 모델 <br>
한국어 텍스트의 감정을 긍정, 부정, 중립 등으로 분류할 수 있도록 설계되어있음


---

## 🃏 게임 규칙
### 이별 사유
1. 답장이 너무 늦을 때
   - 30초 이상 답장이 없으면 이별
2. 성의 없는 단답
   - 응, 그래
3. 의미 없는 반응
   - `ㅋ, ?`가 2개 이상 있는 경우
   - `ㅎ`만 보내는 경우
4. 부정적인 말투
   - 감성 분석 결과 `negative`이거나 일정 점수 이하일 때 부정으로 판단
  
### 해피엔딩 조건
- 100초 이상 버티면 종료
